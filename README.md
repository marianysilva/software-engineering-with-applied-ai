# UNIPDS - Postgraduate Program in Software Engineering with Applied AI, Computer Science

### 🎓 Post-Graduate Program: Applied AI Engineering
**Institution:** UNIPDS / Anhanguera (MEC Certified)  
**Duration:** 12 Months (Starting Feb 2026)  
**Focus:** Integrating LLMs, RAG, and Autonomous Agents into production-ready software systems.

#### 🛠️ Technical Specialization
Unlike theoretical AI courses, this program focuses on the **Applied Engineering** layer—bridging the gap between software development and Generative AI. 

**Key Areas of Study:**
* **Generative AI Architecture:** Working with LLM APIs, Prompt Engineering, and Model Context Protocol (MCP).
* **Agentic Systems:** Developing autonomous agents and integrating AI into existing application servers.
* **AI Operations (AIOps):** Implementing AI tools for DevOps, Security, Governance, and UX/UI.
* **Data Engineering for AI:** Data processing pipelines and Model Fine-Tuning.
* **Capstone Project:** A comprehensive "Integrative Project" applying AI to solve real-world business problems.
